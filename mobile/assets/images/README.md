# Assets

This folder should contain the following images for the FiredUp mobile app:

## Required Images

- **icon.png** (1024x1024) - App icon for iOS and Android
- **adaptive-icon.png** (1024x1024) - Adaptive icon for Android (foreground layer)
- **splash-icon.png** (200x200 or larger) - Icon displayed on splash screen
- **favicon.png** (48x48) - Web favicon

## Design Guidelines

- Use the FiredUp brand color: `#f97316` (orange)
- Keep icons simple and recognizable
- Include fire/flame motif to match the brand

## Generating Icons

You can use tools like:
- [App Icon Generator](https://appicon.co/)
- [Expo CLI icon generation](https://docs.expo.dev/develop/user-interface/app-icons/)
