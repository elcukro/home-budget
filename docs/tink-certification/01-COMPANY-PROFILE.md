## Business Description

FiredUp is a personal finance management SaaS (Software as a Service) application that helps Polish consumers track their income, expenses, savings goals, and loans. The application provides:

1. **Manual Expense Tracking** - Users can manually log their expenses and income
2. **Automatic Bank Integration** - Optional feature to connect bank accounts for automatic transaction import
3. **Budgeting Tools** - Category-based spending analysis and budget planning
4. **Financial Goals** - Baby Steps methodology for debt payoff and savings goals
5. **Loan Management** - Track mortgages, car loans, and other debt

The bank integration feature uses Tink as the Account Information Service Provider (AISP) to securely access account and transaction data from Polish banks.
